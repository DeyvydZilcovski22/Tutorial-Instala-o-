Pré-requisitos:

Caso você não tenha uma hospedagem, utilize o Termux.

Baixe a versão do Termux pelo link:

https://tinyurl.com/2meys6el

Instalação e Configuração Inicial do Termux

1. Atualizar Repositórios:

termux-change-repo

2. Atualizar e Instalar Pacotes Essenciais:

pkg upgrade -y
pkg install nodejs -y
pkg install nodejs-lts -y
pkg install ffmpeg -y
pkg install wget -y
pkg install git -y

3. Configuração do Armazenamento:

termux-setup-storage

Pressione ENTER para confirmar quando solicitado. Caso apareça uma opção com "y", pressione y seguido de ENTER.

Inicialização do Bot

Se o bot estiver na pasta "Download", utilize o seguinte comando para acessar e iniciar:

cd /sdcard/Download/Artorias_System && sh start.sh

Resolução de Erros:

Se houver algum erro relacionado ao módulo "canvas" ou "canva", vá até a pasta node_modules e apague o módulo correspondente.
